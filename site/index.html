Database context without exploratory SQL

DbGraph captures schema metadata and SQL workload structure into local project state, then exposes that context through CLI commands and MCP tools.

Schema snapshots

Capture PostgreSQL and SQLite metadata into `.dbgraph/` with safe defaults and no raw row storage unless explicitly enabled.

Graph search and impact

Search tables, columns, SQL artifacts, inferred relations, and downstream impact paths before editing database-dependent code.

Structured analysis reports

Review security, quality, SQL workload, and performance findings with evidence, confidence, related objects, and suggested fixes.

Quick workflow

Install once, initialize per project, then let agents ask database-structure questions through MCP.

1. Install the CLI `install.sh` or `install.ps1` downloads the matching release binary.
2. Initialize `.dbgraph/` `dbgraph init -i --yes` writes config, snapshots, and instruction folders.
3. Configure your agent `dbgraph install --target codex --yes` writes a managed MCP entry.
4. Snapshot and analyze Run `dbgraph snapshot`, `search`, `impact`, `validate-sql`, and `analyze`.

Use DbGraph from CLI or MCP

DbGraph keeps project-specific state in `.dbgraph/` and exposes stable read-only commands for agent workflows. Project configuration is documented separately so provider setup, profiling, security, and MCP settings are easy to audit. 

dbgraph status
dbgraph search customer --kind table
dbgraph table public.orders
dbgraph relations public.orders --depth 2
dbgraph validate-sql --sql "select * from orders"
dbgraph analyze --scope all --format markdown

Safety defaults

DbGraph is designed for local, read-only database reasoning. Snapshot is the command that connects to the configured database; validation and analysis use local state.

Local-first Project state stays under `.dbgraph/`.
No SQL execution `validate-sql` parses and checks references only.
Masked samples Raw sample storage is off by default.
Secret-aware errors Provider errors avoid printing sensitive connection strings.
